Soco a 80km/h: Conheça o Stomatopoda

Soco a 80km/h

O soco a 80km/h é um ataque que pode ser realizado por um animal, como um cachorro, por exemplo. O ataque é realizado com uma bola de fumaça, que é lançada para o alvo. O alvo é atingido com uma força de 80km/h.
